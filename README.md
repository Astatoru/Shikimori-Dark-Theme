# Темная тема для сайта shikimori.me

Для смены основных цветов, измените эти строчки:

```css 
html[data-color-mode="light"] {
    --clr-main-primary: /* Основной цвет*/;
    --clr-main-dark:/* Темнее цвет*/;
    --clr-main-light:/* Светлее цвет*/;
}
```
